## 2. 平衡膳食准则八条

| 序号 | 准则 | 关键要点 |
|---:|---|---|
| 1 | 食物多样，合理搭配 | 每天摄入 12 种以上食物，每周 25 种以上；谷类为主，粗细搭配。 |
| 2 | 吃动平衡，健康体重 | 天天身体活动；每周至少 5 天中等强度身体活动，累计 150 分钟以上；主动身体活动最好每天 6000 步。 |
| 3 | 多吃蔬果、奶类、全谷、大豆 | 餐餐有蔬菜，天天吃水果；奶类、全谷物、大豆及其制品应经常摄入。 |
| 4 | 适量吃鱼、禽、蛋、瘦肉 | 优先选择鱼和禽；每周最好吃鱼 2 次或 300-500 g；每天一个鸡蛋，不弃蛋黄；少吃深加工肉制品。 |
| 5 | 少盐少油，控糖限酒 | 成人每天食盐 <5 g、烹调油 25-30 g；添加糖每天 <50 g，最好 <25 g；反式脂肪酸 <2 g；成年人如饮酒，酒精量每天不超过 15 g。 |
| 6 | 规律进餐，足量饮水 | 定时定量，不漏餐；每天吃早餐；低身体活动水平成年人每日饮水男性 1700 ml、女性 1500 ml。 |
| 7 | 会烹会选，会看标签 | 规划膳食，选择新鲜、营养素密度高的食物；学会阅读食品标签；在外就餐注意适量与平衡。 |
| 8 | 公筷分餐，杜绝浪费 | 选择新鲜卫生食物；生熟分开；使用公筷公勺，提倡分餐；按需备餐，减少浪费。 |

## 4. 一般人群关键量化建议

| 主题 | 建议 |
|---|---|
| 食物多样 | 每天 12 种以上，每周 25 种以上。 |
| 谷薯类 | 成人每天谷类 200-300 g；全谷物和杂豆 50-150 g；薯类 50-100 g。 |
| 三大产能营养素供能比 | 碳水化合物 50%-65%，蛋白质 10%-15%，脂肪 20%-30%。 |
| 蔬菜 | 每天不少于 300 g，深色蔬菜占 1/2。 |
| 水果 | 每天 200-350 g 新鲜水果，果汁不能代替鲜果。 |
| 奶类 | 每天相当于 300 ml 以上液态奶；膳食宝塔推荐乳制品 300-500 g/d。 |
| 大豆、坚果 | 经常吃全谷物、豆制品，适量吃坚果；宝塔推荐大豆及坚果 25-35 g/d。 |
| 动物性食物 | 成人平均每天 120-200 g；每周鱼类 300-500 g、蛋类 300-350 g、畜禽肉 300-500 g。 |
| 盐、油、糖 | 盐 <5 g/d；烹调油 25-30 g/d；添加糖 <50 g/d，最好 <25 g/d。 |
| 饮酒 | 儿童青少年、孕妇、乳母、慢性病患者不应饮酒；成年人如饮酒，酒精量不超过 15 g/d。 |
| 饮水 | 低身体活动水平成年男性 1700 ml/d，成年女性 1500 ml/d；推荐白水或茶水。 |
| 三餐能量分配 | 早餐 25%-30%，午餐 30%-40%，晚餐 30%-35%。 |
| 运动 | 每周至少 5 天中等强度身体活动，累计 150 分钟以上；每周 2-3 天抗阻运动；减少久坐。 |

## 5. 特定人群关键内容

### 5.1 备孕和孕期妇女

核心推荐：

- 调整孕前体重至正常范围，保证孕期体重适宜增长。
- 常吃含铁丰富的食物，选用碘盐，合理补充叶酸和维生素 D。
- 孕吐严重者少量多餐，保证摄入含必需量碳水化合物的食物。
- 孕中晚期适量增加奶、鱼、禽、蛋、瘦肉摄入。
- 经常户外活动，禁烟酒，保持健康生活方式。
- 积极准备母乳喂养。

| 食物种类 | 备孕/孕早期 | 孕中期 | 孕晚期 |
|---|---:|---:|---:|
| 粮谷类/g | 200-250 | 200-250 | 225-275 |
| 薯类/g | 50 | 75 | 75 |
| 蔬菜类/g | 300-500 | 400-500 | 400-500 |
| 水果类/g | 200-300 | 200-300 | 200-350 |
| 鱼、禽、蛋、肉（含动物内脏）/g | 130-180 | 150-200 | 175-225 |
| 奶/g | 300 | 300-500 | 300-500 |
| 大豆/g | 15 | 20 | 20 |
| 坚果/g | 10 | 10 | 10 |

### 5.2 哺乳期妇女

核心推荐：

- 产褥期食物多样不过量，坚持整个哺乳期营养均衡。
- 适量增加富含优质蛋白质及维生素 A 的动物性食物和海产品，选用碘盐，合理补充维生素 D。
- 家庭支持，愉悦心情，充足睡眠，坚持母乳喂养。
- 增加身体活动，促进产后恢复健康体重。
- 多喝汤和水，限制浓茶和咖啡，忌烟酒。

乳母一天食物建议量：

| 食物种类 | 建议量 |
|---|---:|

| 全谷物和杂豆 | 不少于 1/3 |
| 薯类 | 75 g |
| 蔬菜类 | 400-500 g |
| 其中绿叶蔬菜、红黄色等有色蔬菜 | 占 2/3 以上 |
| 水果类 | 200-350 g |
| 鱼、禽、蛋、肉（含动物内脏） | 175-225 g |
| 牛奶 | 300-500 ml |
| 大豆类 | 25 g |
| 坚果 | 10 g |
| 烹调油 | 25 g |
| 食盐 | <5 g |
| 饮水量 | 2100 ml |
| 动物肝脏 | 每周 1-2 次，总量约 85 g 猪肝或 40 g 鸡肝 |

### 5.3 0-6 月龄婴儿

核心推荐：

- 母乳是婴儿最理想的食物，坚持 6 月龄内纯母乳喂养。
- 生后 1 小时内开奶，重视尽早吸吮。
- 顺应喂养，建立良好的生活规律。
- 适当补充维生素 D，母乳喂养无需补钙。
- 若有动摇母乳喂养的想法或需改变喂养方式，应咨询医务人员。
- 定期监测婴儿体格指标，保持健康生长。

### 5.4 7-24 月龄婴幼儿

核心推荐：

- 继续母乳喂养，满 6 月龄起添加辅食。
- 及时引入多样化食物，重视动物性食物添加。
- 婴幼儿辅食单独制作；少加糖、盐和调味品，保持食物原味；辅食应含适量油脂。
- 提倡回应式喂养，鼓励但不强迫进食。
- 注重饮食卫生和进食安全。
- 每 3 个月测量一次身长、体重、头围等体格生长指标，追求平稳生长。

辅食添加要点：

| 主题 | 要点 |
|---|---|
| 起始时间 | 满 6 月龄开始添加辅食；不应在满 4 月龄前添加。 |
| 第一口辅食 | 从富含铁的泥糊状食物开始，如肉泥、蛋黄、强化铁婴儿米粉。 |
| 食物引入 | 每次只引入一种新食物，由少到多、由稀到稠、由细到粗，适应 2-3 天后再添加新食物。 |
| 过敏食物 | 不盲目回避易过敏食物，1 岁内适时引入各种食物。 |
| 进餐环境 | 进餐时不看电视、不玩玩具，每次进餐时间不超过 20 分钟。 |

### 5.5 学龄前儿童（2-5 岁）

核心推荐：

- 食物多样，规律就餐，自主进食，培养健康饮食行为。
- 每天饮奶，足量饮水，合理选择零食。
- 合理烹调，少调料、少油炸。
- 参与食物选择与制作，增进对食物的认知和喜爱。
- 经常户外活动，定期体格测量，保障健康成长。

关键量化建议：

| 主题 | 建议 |
|---|---|
| 餐次 | 每天三次正餐和两次加餐，两正餐间隔 4-5 小时，加餐与正餐间隔 1.5-2 小时。 |
| 食物多样 | 平均每天 12 种以上，每周 25 种以上。 |
| 奶类 | 每天 300-500 ml 或相当量奶制品。 |
| 总水摄入 | 1300-1600 ml/d，其中饮水量 600-800 ml，以白水为佳。 |
| 户外活动 | 每天至少 120 分钟。 |
| 视屏时间 | 每天累计不超过 1 小时，越少越好；每次久坐不超过 1 小时。 |
| 睡眠 | 每天 10-13 小时，其中包括 1-2 小时午睡。 |
| 食盐 | 2-3 岁 <2 g/d，4-5 岁 <3 g/d。 |

### 5.6 学龄儿童（6-17 岁）

核心推荐：

- 主动参与食物选择和制作，提高营养素养。
- 吃好早餐，合理选择零食，培养健康饮食行为。
- 天天喝奶，足量饮水，不喝含糖饮料，禁止饮酒。
- 多户外活动，少视屏时间，每天 60 分钟以上中高强度身体活动。
- 定期监测体格发育，保持体重适宜增长。

关键补充：

| 主题 | 建议 |
|---|---|
| 早餐 | 早餐应包括谷薯类、蔬菜水果、奶、动物性食物、豆、坚果等食物中的 3 类及以上。 |
| 奶类 | 每天约 300 g 液态奶或相当量奶制品。 |
| 身体活动 | 每天累计至少 60 分钟中高强度身体活动；每周至少 3 天高强度身体活动；每周至少 3 天增强肌肉或骨健康的运动。 |
| 视屏时间 | 每天限制在 2 小时内。 |

### 5.7 老年人

一般老年人（65-79 岁）核心推荐：

- 食物品种丰富，动物性食物充足，常吃大豆制品。
- 鼓励共同进餐，保持良好食欲，享受食物美味。
- 积极户外活动，延缓肌肉衰减，保持适宜体重。
- 定期健康体检，测评营养状况，预防营养缺乏。

关键量化建议：

| 主题 | 建议 |
|---|---|
| 动物性食物 | 平均每日 120-150 g，其中鱼 40-50 g、畜禽肉 40-50 g、蛋类 40-50 g。 |
| 奶类 | 每日 300-400 ml 牛奶或相当蛋白质含量的奶制品。 |
| 大豆 | 平均每天相当于 15 g 大豆。 |
| 蛋白质 | 一般每日 1.0-1.2 g/kg 体重；进行抗阻训练者可 >1.2-1.5 g/kg 体重。 |
| 适宜 BMI | 老年人适宜 BMI 范围约 20.0-26.9 kg/m²。 |

高龄老年人（80 岁及以上）核心推荐：

- 食物多样，鼓励多种方式进食。
- 选择质地细软、能量和营养素密度高的食物。
- 多吃鱼、禽肉、蛋类、奶制品及大豆类等高营养价值食物。
- 关注体重丢失和进食不足；如膳食摄入不足目标量的 80%，应在医生和临床营养师指导下适时补充营养。
- 坚持健身与益智活动，促进身心健康。

### 5.8 素食人群

核心推荐：

- 食物多样，谷类为主，适量增加全谷物。
- 增加大豆及其制品摄入，选用发酵豆制品。
- 常吃坚果、海藻和菌菇。
- 蔬菜、水果应充足。
- 合理选择烹调油。
- 定期监测营养状况。

成年素食者推荐膳食组成：

| 食物种类 | 全素 | 蛋奶素 |
|---|---:|---:|
| 谷类/g | 250-400 | 225-350 |
| 其中全谷物和杂豆/g | 120-200 | 100-150 |
| 薯类/g | 50-125 | 50-125 |
| 蔬菜/g | 300-500 | 300-500 |
| 其中菌藻类/g | 5-10 | 5-10 |
| 水果/g | 200-350 | 200-350 |
| 大豆及其制品/g | 50-80 | 25-60 |
| 其中发酵豆制品/g | 5-10 | - |
| 坚果/g | 20-30 | 15-25 |
| 烹调油/g | 20-30 | 20-30 |
| 奶/g | - | 300 |
| 蛋/g | - | 40-50 |
| 食盐/g | 5 | 5 |

素食者容易不足的营养素及主要来源：

| 营养素 | 主要来源 |
|---|---|
| n-3 多不饱和脂肪酸 | 亚麻籽油、紫苏油、核桃油、大豆油、菜籽油、奇亚籽油、部分藻类 |
| 维生素 B12 | 发酵豆制品、强化食品或补充剂 |
| 维生素 D | 强化维生素 D 的食物、晒太阳 |
| 钙 | 大豆、芝麻、海带、黑木耳、绿色蔬菜；蛋奶素可摄入奶及奶制品 |
| 铁、锌 | 全谷物、大豆、坚果、菌菇类 |
